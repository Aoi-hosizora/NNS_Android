# NNS_Android
+ Developed by [Flutter](https://flutter.dev/) framework

### Environment
+ `Flutter 1.5.4-hotfix.2`
+ `Dart 2.3.0`
+ `VS Code`
+ (`AS 3.3.x`)

### Build & Run
```bash
flutter packages get # Get dependencies from pubspec

adb connect 127.0.0.1:xxxxx # Connect adb server
flutter run # Run App through adb
```
